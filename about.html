About DS-Algo Playground

DS-Algo Playground is an interactive educational website for learning Data Structures and Algorithms. It combines visual animations, multi-language code examples, and knowledge quizzes to create an engaging study experience.

How to Use

  • Browse Topics — Use the sidebar or home page grid to navigate to any topic.
  • Watch Visualizations — Use play/pause, step forward/backward, and speed controls to understand how algorithms work step-by-step.
  • Read Code — Toggle between Python, Java, and C++ implementations. Click "Copy" to grab the code.
  • Take Quizzes — Test your understanding with multiple-choice questions. Your scores are saved locally.
  • Track Progress — The progress bar on the home page shows your overall completion across all topics.
  • Use Shortcuts — Press ? to view all keyboard shortcuts.

Tech Stack

  • Vanilla HTML, CSS, and JavaScript — no frameworks or build tools
  • Canvas API for all visualizations
  • CSS Custom Properties for dark/light theming
  • localStorage for progress tracking
  • Regex-based syntax highlighting
  • Hosted on GitHub Pages

Topics Covered

  • Data Structures — Arrays, Linked Lists, Stack, Queue, Hash Table, Binary Search Tree, Heap
  • Sorting — Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort
  • Searching — Binary Search, BFS, DFS
  • Techniques — Two Pointers, Sliding Window, Dynamic Programming, Recursion
